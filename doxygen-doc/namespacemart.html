Detailed Description

Define the class Mutation which implements the brain of the mutation.

Define the class UserMaps which is used to map operation name (as string) with the corresponding opcode (expElemKeys). it also map each opcode to the corresponding mutation op's C++ class object.

Define the classes and types used by mutation process.

Define the class TCE which is used to call optimizer and code diff.

Generic abstract base class for replacors (operation that are not matched, but replace others - like Abs, delStmt).

Class for replacing by a trap. Actually inexistant. Please use the method.

Class for replacing shuffling switch's case and default destination basic blocks. Actually inexistant. only the class Switchcases can implements and use such replacer.

Class for shuffling function call arguments of same type. Actually inexistant. only the class FunctionCall can implements and use such replacer.

Class for removing some switch's cases, thus making it them point to default basic block. Actually inexistant. only the class SwitchCases can implements and use such replacer.

Class for replacing function call callee. Actually inexistant. only the class FunctionCallCallee can implements and use such replacer.

Class for replacing an expression with one of its operator (passed in the first operand of the method createReplacement())

Class for replacing a numeric integer expresion with absolute value.

Class for replacing a numeric Floating Point expresion with absolute value.

Class for replacing by deletion. Actually inexistant. Please use the method.
