◆ doMutate()

bool Mutation::doMutate ( )

This class create a proxy BB for each incoming BB of PHI nodes whose corresponding value is a CONSTANT (independent on value computed on previous BB)

The incoming value is a constant

This hel recording the IR's LOC: index in the function it belongs

In case we have multiBB stmt, this say which is the first BB to start mutation from. this is equal to itBBlock below if only sigle BB stmts Set to null after each actual mutation take place

Do not mutate the inserted proxy for PHI nodes

set the Basic block from which the actual mutation should start

Actual mutation **********************************

mutate all the basic blocks between 'mutationStartingAtBB' and '&*itBBlock'

Get all the mutants

Do not mutate the inserted proxies for PHI nodes

If the firstInst (intended basic block plit point) isPHI Node, instead of splitting, directly add the mutant selection switch on the Proxy BB.

Mutation over for the current set of BB, reinitialize 'mutationStartingAtBB' for the coming ones
